# Estudo de `<div>` e `<span>` em HTML

Projeto pra entender a diferença entre elemento de bloco (`<div>`) e elemento inline (`<span>`).

## Conceitos estudados

- **`<div>`** — pra dividir a página em seções (header, banner, produtos, rodapé). Cada div ocupa a linha inteira.
- **`<span>`** — pra estilizar partes específicas do texto sem quebrar a linha, tipo o telefone no header e o "Magazine Luiza" no rodapé.
